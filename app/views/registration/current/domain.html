Check availability of a .gov.uk domain name

A .gov.uk domain name must clearly describe your registrant's organisation or government initiative. For example a:

  • central department can use hmrc.gov.uk
  • county council can use northamptonshire.gov.uk
  • town council can use biggleswadetowncouncil.gov.ukor biggleswadetown.gov.ukor biggleswade-tc.gov.uk
  • parish councils can use kirkbymalzeardparishcouncil.gov.uk, kirkbymalzeardparish.gov.uk, kirkbymalzeard-pc.gov.uk

If your parish council uses an established abbreviation in its current domain, you may be able to use it for a .gov.uk domain. Any domain applications using an abbreviation must end with -pc.gov.uk, -parish.gov.uk,
-parishcouncil.gov.uk or -tc.gov.uk, -town.gov.uk, -towncouncil.gov.uk. For example, kmld-pc.gov.ukor bgw-tc.gov.uk

We approve domains on a first come first serviced basis.
